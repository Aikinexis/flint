This test verifies that context is assembled in the correct priority order for all AI operations.

Expected Context Priority Order

1 CURRENT DATE AND TIME - Temporal context (highest priority)
2 DOCUMENT TITLE / PROJECT CONTEXT - Document structure
3 DOCUMENT CONTENT / CONTEXT - Surrounding text, related sections
4 AUDIENCE AND TONE GUIDANCE - Pinned notes
5 USER'S INSTRUCTION - Specific directive (most specific)
