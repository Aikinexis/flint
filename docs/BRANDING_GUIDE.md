# Flint - Branding & Extension ID Guide

## Understanding Chrome Extension IDs

### What is the Extension ID?

The extension ID (`occpmnpeaaiobnkkgkcmbgcigocnpcjh`) is a **cryptographic hash** automatically generated by Chrome based on your extension's private key. It **cannot be customized** or changed to a readable name.

### Why Can't I Change It?

Chrome uses this ID to:
1. Uniquely identify your extension
2. Maintain security and prevent spoofing
3. Track updates and versions
4. Manage permissions and storage

### When Does It Change?

The ID only changes if you:
1. Delete and recreate the extension from scratch
2. Generate a new private key
3. Publish to Chrome Web Store (gets a new stable ID)

**Important**: Once published to Chrome Web Store, the ID becomes permanent and tied to your developer account.

## Extension ID Reference

### Development ID
```
occpmnpeaaiobnkkgkcmbgcigocnpcjh
```

**Note**: This ID is only for local development. When you publish to Chrome Web Store, you'll get a new permanent ID.

### How to Find Your Extension ID

1. Open `chrome://extensions/`
2. Enable "Developer mode" (top right)
3. Find your extension
4. The ID is shown below the name

### Using the Extension ID in Code

```typescript
// Get current extension ID dynamically
const extensionId = chrome.runtime.id;

// Never hardcode the ID - it changes between dev and production
```
